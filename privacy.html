Demo privacy

Privacy notice

This is a static demo page created for presentation purposes. Form data is stored locally in the browser using localStorage and is not sent to any server.

Demo purpose

This website is a presentation demo for IAN in the context of Web Summit Rio 2026. It is designed to show a possible referral flow and visual direction.

Local storage

When the referral form is submitted, the demo stores the latest referral summary in the current browser only. Clearing browser data removes the stored information.

No backend

There is no server, database, CRM integration or external form submission in this demo. The form exists only to simulate the experience.
